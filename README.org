This is a personal repository for experiment with the idea of [[https://specifyapp.com/blog/becoming-a-design-tokens-ambassador][design tokens pipeline]].
It receives design tokens from [[https://www.figma.com/community/plugin/888356646278934516/Design-Tokens][Design Tokens plugin for Figma]] via a webhook.
It is basically a stripped minimal version of [[https://github.com/lukasoppermann/design-token-transformer][lukasoppermann/design-token-transformer]].
